# gollum

主要目用于更好的理解DDD+CQRS，以及思考如何更容易的让DDD落地

## gollum-common

封装了一些通用功能

## gollum-core

实现了事件驱动的DDD+CQRS+ES框架核心

## gollum-simple

实现了简单的DDD+CQRS框架核心, 能很方便的在小项目中也应用DDD思想

## sample-note

基于gollum-core和spring的一个简单记事本demo

## sample-bank

基于gollum-core和spring的一个银行转账demo
